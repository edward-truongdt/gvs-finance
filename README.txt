HƯỚNG DẪN SỬ DỤNG NHANH
------------------------

1. CHẠY ỨNG DỤNG
   - Double-click file: GVS_Finance_Data.exe
   - Chờ vài giây để ứng dụng khởi động
   - Giao diện chính sẽ hiện ra

2. DOWNLOAD DỮ LIỆU
   - Click nút "Add Symbols" để thêm mã chứng khoán
   - Nhập mã (VD: AAPL, BTC-USD, VNM.VN)
   - Click "Download" để tải dữ liệu
   - Xem progress bar và log

3. SỬ DỤNG DỮ LIỆU
   - Sau khi download xong
   - Import vào các phần mềm phân tích và sử dụng dữ liệu

YÊU CẦU HỆ THỐNG
-----------------
✓ Windows 10/11 (64-bit)

✓ ~150 MB dung lượng ổ đĩa

TÍNH NĂNG
---------
✓ Download từ Yahoo Finance (stocks, crypto, futures, ETFs)
✓ Hỗ trợ nhiều markets (US, VN, crypto, etc.)
✓ Cache tự động (tải 1 lần, dùng nhiều lần)
✓ Export sang định dạng MetaStock
✓ Multi-threaded downloads (nhanh)
✓ Timezone tự động (15+ markets)

TROUBLESHOOTING
---------------

LỖI: "Windows protected your PC"
→ Click "More info" → "Run anyway"

LỖI: "Missing DLL"
→ Cài Visual C++ Redistributable:
  https://aka.ms/vs/17/release/vc_redist.x64.exe

LỖI: Antivirus block
→ Add exception cho GVS_Finance_Data.exe

LỖI: Download fails
→ Kiểm tra kết nối Internet
→ Thử lại sau vài phút (Yahoo Finance rate limit)

ỨNG DỤNG CHẠY CHẬM
→ Lần đầu tiên có thể chậm (extract files)
→ Lần sau sẽ nhanh hơn
